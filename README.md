# DeWord 高效私人背单词

DeWord 是一个面向 Windows 桌面端的个人背单词软件原型，核心目标是高效、少干扰、可长期坚持。界面参考用户提供的设计稿，保留了今日单词、复习计划、词库、错词本、统计和设置，删除了社交、排行榜等非必要功能。

## 记忆策略

软件按以下学习证据设计：

- **主动回忆优先**：先隐藏释义、例句和词源，要求用户在脑中回忆后再查看答案，避免“看着眼熟”的错觉。
- **间隔重复**：根据“认识 / 模糊 / 不认识”的自评动态调整下次复习日期，常见间隔为 1、3、7、15、30 天。
- **交错复习**：新词、错词和到期复习混排，减少短时间内机械重复带来的虚假熟练。
- **即时反馈**：答题后显示例句、中文释义和词源钩子，帮助同时建立 form、meaning、use 三类词汇知识。
- **短时每日目标**：默认每天 12 个新词和 20 个复习词，优先保证连续学习，而不是一次性刷大量单词。

## 已实现功能

- Tauri + React + TypeScript 桌面应用骨架，可打包为 Windows 桌面端。
- 今日单词卡片：隐藏答案、查看释义、三档评分、自动排期。
- 复习计划：今日任务、复习间隔分布、预计用时。
- 词库管理、错词本、学习统计和设置页的完整 UI 原型。
- 响应式布局，窗口较窄时自动收起侧边栏文字。

## 研究参考

- Retrieval Practice 的资料强调，把学习分散到多个短会话并结合提取练习，比集中突击更利于长期保持。
- Indiana University CITL 对 spaced practice 的定义是：学习发生在多个时间点，中间留有休息间隔，而不是单次集中学习。
- MIT Open Learning 总结指出，跨时间间隔练习通常比把练习集中在一个会话中更有利于学习。
- 二语词汇学习相关研究常把间隔重复、交错练习和词汇的 meaning/form/use 多维知识结合起来设计软件。
